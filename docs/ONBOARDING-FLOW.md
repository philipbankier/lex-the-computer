# Lex the Computer — Onboarding Flow

Based on Zo's onboarding (from Philip's screenshots), here's what we replicate:

## Step 1: Welcome Screen
- "Welcome to Lex" with the logo
- "Your personal AI cloud computer"
- [Get Started] button

## Step 2: Create Account
- Email + password (or magic link)
- Choose your handle (becomes handle.lex.space)
- This provisions their container in the background

## Step 3: Tell Lex About Yourself
- Name
- What you do (free text bio)
- Interests / areas of focus
- Social links (optional — for Space page)
- "This helps Lex personalize responses for you"

## Step 4: Choose Your AI's Personality
- Grid of persona cards:
  - **Default** — Balanced, helpful, concise
  - **Creative** — Imaginative, exploratory, verbose
  - **Technical** — Precise, code-oriented, minimal
  - **Casual** — Friendly, conversational, emoji-friendly
  - **Custom** — Write your own
- User picks one (can change later in Settings)

## Step 5: Set Up Your First Automation
- "Lex can work for you even when you're away"
- Suggested automations:
  - 📰 Daily news briefing on topics you care about
  - 📧 Morning email summary
  - 📅 Daily calendar preview
  - 🔔 Website monitor (alert when something changes)
  - ⏭️ Skip for now
- User picks one → fills in details (e.g., "What topics?" for news)
- Sets delivery: "Send to chat" / "Email me" / "Text me"

## Step 6: Connect Channels (Optional)
- "Access Lex from anywhere"
- Telegram: [Connect Telegram] → QR code or pairing code
- Email: "Your Lex email is handle@lex.computer" (auto-provisioned)
- Discord: [Connect Discord]
- Skip for now

## Step 7: Ready!
- "Your computer is ready 🖥️"
- Show the user's Space URL: handle.lex.space
- Show their Lex email: handle@lex.computer
- Quick action buttons:
  - [Start Chatting] → go to chat
  - [Upload Files] → go to file browser
  - [Build a Site] → go to sites
  - [Explore Settings] → go to settings

## Post-Onboarding: First Chat
- Lex greets the user with context from their bio
- "Hey [name]! I'm Lex, your personal AI computer. I've set up your daily [automation] — it'll run every morning at 8am. What would you like to work on?"
- This mirrors what Zo does in Philip's screenshots
